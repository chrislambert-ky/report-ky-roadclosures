About This Project

This project provides an interactive report for analyzing road closures in Kentucky. It includes various visualizations and dashboards to help users understand closure patterns and durations.

Data Pipeline:

  1. Step 1: Fetch data from CSV source, parse as JSON, and save it to data_full_dataset.json.
  2. Step 2: Fetch, parse, and filter fields, and save to data_parsed_data.json.
  3. Step 3: Fetch, parse, filter, and geo-enrich using the KYTC API, and store as data_reportready_dataset.json.
